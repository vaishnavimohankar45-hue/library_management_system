# Library Management System
## Description
This is a DBMS project developed using MySQL to manage library operations like students, books, and issue/return records.

## Features
- Student Management
- Book Management
- Issue and Return System
- SQL queries with relationship

## Technologies Used
- MySQL
- SQL
- MySQL Workbench

  ## Database Tables
  - Students
  - Books
  - Librarian
  - Issue

    ## How to Run
    1. Open MySQL Workbench
    2. Import 'library_management_system.sql' file
    3. Run all queries
    4. Check tables using: sql SHOW TABLES;
